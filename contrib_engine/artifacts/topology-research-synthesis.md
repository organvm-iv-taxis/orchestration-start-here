## I. The Hegemonizing Swarm Trope (Fiction → Architecture)

The "face of machines" from Matrix Revolutions (Deus Ex Machina) is a **granular system / swarm robotics forming a volumetric display** — thousands of discrete autonomous agents forming a reconfigurable 3D shape. The trope name is "Hegemonizing Swarm" or "Programmable Matter."

### Fiction Catalog (mapped to topology types)

| Source | Formation Behavior | Topology Analogue | What We Can Steal |
|--------|-------------------|-------------------|-------------------|
| **Deus Ex Machina** (Matrix) | Insect-sized units form a face — volumetric display | FISSION_FUSION → morphing into a macro-shape | Shape-from-function: the face IS the communication interface |
| **Microbots** (Big Hero 6) | Magnetic actuators link to form bridges, towers, hands via neural headband | STAR (headband = hub) → MESH (linked formation) | Central controller that dissolves into distributed mesh |
| **Prey nano-swarm** (Crichton) | Predator-prey evolution, learns topology through adversarial pressure | SWARM → emergent HIERARCHY through generational selection | Fitness-based topology selection (#59) |
| **Replicators** (Stargate SG-1) | Nanites that look human, dissolve to dust, reshape limbs to weapons | STIGMERGIC → FISSION (dissolve) → FUSION (reform as weapon) | Mode switching: passive form vs. active weapon form |
| **KSI Drones** (Transformers: AoE) | Break into floating cubes, swarm through air, reassemble | MESH → SWARM (airborne cubes) → new MESH (reassembled) | Zero-cost reconfiguration — no explicit "unbond" step |
| **The Invincible** (Lem, 1964) | Crystal micro-machines form electromagnetic intelligence | STIGMERGIC (individual = mindless) → SWARM (collective = intelligent) | Intelligence emergence threshold — N agents needed for cognition |
| **Blood Music** (Bear) | Modified cells reorganize host body from inside | RHIZOMATIC (cells are laterally connected, no hierarchy) | The substrate IS the agent network — no separation between infrastructure and computation |
| **Kilobots** (Harvard) | 1000 coin-sized robots self-assemble into shapes via local vibration/communication | STIGMERGIC (local-only communication) → macro-shape | Proven in hardware: N=7-style local interaction produces global shape |
| **CoCoRo underwater swarm** | Drone swarm acts like school of fish, shares information locally | MESH (fluid) → "flow around obstacles like a liquid" | Amorphous topology — no named formation, continuous flow |

### Key Insight: Formation = Communication Interface

The Matrix face is not decorative — it's the swarm's communication interface with Neo. The formation IS the function. This maps directly to our topology model: the shape an agent group takes is not aesthetic, it's the group's way of presenting its collective capability to the outside world.
